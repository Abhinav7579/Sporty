# Sporty

## Project Description
Sporty is an innovative web application designed to bring sports enthusiasts together. It offers a platform for users to find teams, join events, and stay updated on their favorite sports activities. The app's sleek interface and user-friendly features make engaging with sports easier and more fun.

## Tech Stack
- **Node.js**: A powerful JavaScript runtime built on Chrome's V8 JavaScript engine, allowing for efficient server-side development.
- **Drizzle ORM**: An Object-Relational Mapping tool for Node.js and TypeScript that simplifies database interactions, enabling developers to work intuitively with their data models.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Abhinav7579/Sporty.git
   cd Sporty
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up environment variables**: Create a `.env` file in the root of the project and add your configurations based on the provided `.env.example`. 
4. **Run the application**:
   ```bash
   npm start
   ```

## Usage Guidelines
- Visit the home page to explore upcoming sports events.
- Sign up for an account to join events and connect with other users.
- Use the user dashboard to manage your profile and view your activities.

Feel free to contribute to the project by opening issues or submitting pull requests!
